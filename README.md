# Stackline Dashboard

## Project Setup

1. Clone the repository:
    ```sh
    git clone <repo-url>
    cd stackline-dashboard
    ```
2. Install dependencies:
    ```sh
    npm install
    ```
3. Run the project:
    ```sh
    npm run dev
    ```

## Features

- React + TypeScript + Redux
- TailwindCSS for styling


## Deployment

Deployed via GitHub Pages / Vercel / Netlify.
